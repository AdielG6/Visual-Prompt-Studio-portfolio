Assignment title: what is a cyanotype?

A cyanotype is a collection of shapes and objects joined together to create one mark created by sunlight and shadow and then washing it to create a big shapes coming out as white, while the background is blue.

The way how it changes the subject for cyanotype based from capturing sunlight is based from the different shapes and structures needed to print into one big shape if left laying on top of the sheet for a while leaving marks to create a shape.
